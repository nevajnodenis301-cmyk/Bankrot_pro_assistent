# Шаблоны документов

## bankruptcy_application.docx

Шаблон заявления о признании банкротом физического лица.

### Плейсхолдеры для docxtpl:

- `{{ case_number }}` - номер дела
- `{{ full_name }}` - ФИО должника
- `{{ birth_date }}` - дата рождения
- `{{ passport }}` - серия и номер паспорта
- `{{ inn }}` - ИНН
- `{{ address }}` - адрес регистрации
- `{{ total_debt }}` - общая сумма долга
- `{{ creditors_count }}` - количество кредиторов
- `{% for creditor in creditors %}` - цикл по кредиторам
  - `{{ creditor.name }}` - название кредитора
  - `{{ creditor.debt_amount }}` - сумма долга кредитору
  - `{{ creditor.debt_type }}` - тип долга
- `{{ current_date }}` - текущая дата

### Создание шаблона:

1. Создайте документ Word (.docx)
2. Вставьте плейсхолдеры в нужные места
3. Сохраните как `bankruptcy_application.docx`
4. Поместите в директорию `api/templates/`

### Пример структуры документа:

```
                            В Арбитражный суд [название]

                            От: {{ full_name }}
                            Дата рождения: {{ birth_date }}
                            Паспорт: {{ passport }}
                            ИНН: {{ inn }}
                            Адрес: {{ address }}

                    ЗАЯВЛЕНИЕ
            о признании гражданина банкротом

Дело № {{ case_number }}

Я, {{ full_name }}, не могу исполнить денежные обязательства перед кредиторами.

Общая сумма задолженности составляет: {{ total_debt }} рублей.

Кредиторы ({{ creditors_count }}):

{% for creditor in creditors %}
{{ loop.index }}. {{ creditor.name }} - {{ creditor.debt_amount }} руб. ({{ creditor.debt_type }})
{% endfor %}

На основании изложенного и руководствуясь статьями 213.1-213.9 Федерального закона
"О несостоятельности (банкротстве)" от 26.10.2002 № 127-ФЗ,

ПРОШУ:
1. Признать меня банкротом
2. Ввести процедуру реструктуризации долгов

Дата: {{ current_date }}                             _____________ {{ full_name }}
```

Для полноценной работы необходимо создать реальный файл .docx с помощью Microsoft Word или LibreOffice.
Данный README содержит только описание структуры.
